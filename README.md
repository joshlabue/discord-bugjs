# discord-bugjs
A Discord bot that helps tracks bugs in your crappy code.

# Requirements
* Node > v6.0.0
* Discord
* A somewhat functioning computer with access to the internet

discordapp.com/oauth2/authorize?client_id=(ID OF YOUR BOT)&scope=bot&permissions=0
